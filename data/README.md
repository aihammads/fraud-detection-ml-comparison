# 📊 Dataset

This project utilizes the **Credit Card Fraud Detection Dataset**, a widely used benchmark for fraud detection in imbalanced financial data.

📥 Download from:
https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud

## 📥 Instructions

1. Download `creditcard.csv`
2. Place it in this folder:

`data/creditcard.csv`

## ⚠️ Note

The dataset is not included in this repository due to file size limitations.
